2.2 选择合适的地区后，进入 App Launchpad > Create App
2.3 填写配置参数：

  • Application name:
    nazhamb

  • Imgage name:
    ghcr.io/nezhahq/nezha:v1.13.2

  • Network:
    端口：8080
    自定义域名：nz.363689.xyz

  • Local Storage:

    • 点add
    • Capacity：
      1或2
    • Mount Path：
      /dashboard/data

2.4启动部署，等状态为 Running 即可
